# Мультимедийная программа к выставке «Петербург декабристов»

## Описание проекта
Поскольку данная программа является коммерческим проектом, предоставлю сжатое описание проекта согласно ТЗ от заказчика:  

Цель - создание мультимедийной программы с разработкой мультимедийного контента для интерактивной карты. 

Содержание мультимедийной программы к выставке «Петербург декабристов» в комеданстком доме Петропавловской крепости.  
Раздел 1. Памятные места декабристов в Петербурге, включая места проживания   
Раздел 2. Памятные доски, посвященные декабристам и памятным местам, сохранившиеся в Петербурге  
Раздел 3. Сводный список декабристов, сидевших в Петропавловской крепости, с краткими биографиями и адресами в СПб  
Раздел 4. Топография по Петропавловской крепости с местами указаниями камер  
Раздел 5. Форма и основная атрибутика полков Российской императорской армии  
Раздел 6. Цитаты, воспоминания декабристов  

Мультимедийные программы позволят посетителю выставки ознакомиться не только с памятными местами, связанными с декабристами, но и с топографией места их заключения.   
В мультимедийную программу включена интерактивная карта Санкт-Петербурга с отметками зданий, связанных с жизнью и деятельностью декабристов.  
При клике на объект карты, должен происходить переход на соответствующий объект мультимедийного проекта.   
Аннотационный текст размещается рядом с соответствующим изображением.  

Дизайн ПО должен быть простым и интуитивно понятным, эстетически привлекательным, интерактивным, комбинировать информативный и развлекательный стили подачи контента.  
ПО должно предоставлять возможность самостоятельно управлять контентом, включая
-  использование динамичной и статичной компьютерной графики;  
-  наличие музыкального сопровождения, где необходимо;  

Мультимедийная программа должна включать в себя документы и изобразительный материал, а также сопроводительный текст.  
Создание мультимедийных программ должно учитывать:  
-  возможность демонстрации как со звуковым сопровождением, так и без него;  
-  использование динамичной и статичной компьютерной графики;  
-  наличие музыкального сопровождения, где необходимо;  

### Используемое ПО
Основное ПО, на котором предполагается построить общее программное решение комплекса, должно обладать следующими характеристиками:
-  допускать неограниченное наращивание комплексов устройств отображения;  
-  устанавливаться на стандартные компьютеры;  
-  поддерживать динамические изображения HTML5.  

Оборудование (сенсорный монитор Iliyama 19 дюймов и мини ПК) для инсталляции разработанной программы предоставляется Заказчиком.  
Эксплуатация будет производится в формате интерактивной панели.   

## Ход работы
В рамках работы над проектом использовались следующиее ПО/инструменты:
* TypeScript + React - логика работы
* HTML, CSS - верстка и стилизация страниц
* npm - пакетный менеджер

Сборка и запуск: `npm install && npm build && npm run electron-start`
