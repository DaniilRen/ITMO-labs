Лабораторная работа: Вычисление Жордановой нормальной формы матрицы

Цель работы: Изучить и реализовать на практике алгоритм приведения квадратной матрицы к Жордановой нормальной форме. Познакомиться с проблемами точных и численных вычислений в линейной алгебре, а также с методами их решения с использованием матричных разложений (SVD) и библиотек символьной алгебры.

Общие требования и ограничения

  • Язык программирования: Любой (Python, C++, Java и др.).
  • Использование библиотек: Разрешено использовать базовые возможности библиотек линейной алгебры (например, NumPy в Python, Eigen в C++, Apache Commons Math в Java) для хранения матриц и выполнения базовых операций.
  • Разрешенные встроенные операции:
    • Сложение, вычитание и умножение матриц/векторов.
    • Транспонирование матриц.
    • Нахождение определителя (детерминанта).
    • Нахождение ранга матрицы.
    • Нахождение обратной матрицы.
    • Алгоритм Гаусса (для решения СЛАУ или приведения к ступенчатому виду).
  • Запрещенные встроенные операции: Встроенные функции для поиска собственных значений, собственных векторов, сингулярного разложения (для пункта 3) и готовые функции вычисления ЖНФ (например, scipy.linalg.jordan_form или Matrix.jordan_form() из SymPy) при реализации ядра алгоритма использовать нельзя. Вся логика построения жордановых цепочек и базисов должна быть написана вами самостоятельно.

Обязательная часть

Поиск ЖНФ для матриц с целочисленным спектром

В данной части предполагается, что на вход подается матрица $A \in \mathbb{Z}^{n \times n}$, все собственные значения которой являются целыми числами ($\lambda_i \in \mathbb{Z}$).

Задачи:

  1. Поиск собственных значений: Так как корни целые, реализовать поиск корней многочлена можно путем перебора делителей свободного члена.
  2. Поиск корневых подпространств и жордановых цепочек:
    • Для каждого собственного значения $\lambda$ вычислить дефекты матриц $(A - \lambda I)^k$ для определения размеров жордановых клеток.
    • Реализовать алгоритм построения базиса из обобщенных собственных векторов (решая СЛАУ методом Гаусса).
  3. Вы должны выдавать саму Жорданову матрицу $J$ и матрицу перехода $P$, такую что $A = P J P^{-1}$.

Требования к коду:

  • Подготовить скрипт демонстрации работоспособности кода, который прогоняет алгоритм на нескольких заранее заготовленных матрицах (включая диагонализуемые матрицы, нильпотентные матрицы и матрицы с одной жордановой клеткой максимального размера) и выводит результаты.

Продвинутая часть выполняет по желанию. Выполнение продвинутой части дает 10 баллов автоматом за 2-ую контрольную работу.

Продвинутый уровень A

Использование символьной алгебры для алгебраических чисел

Матрицы с целыми элементами не всегда имеют целые или даже рациональные собственные значения. В этом задании спектр матрицы состоит из алгебраических чисел. Численные погрешности здесь недопустимы, алгоритм должен работать в точной арифметике.

Задачи:

  1. Подключить к проекту библиотеку для символьных вычислений.
  2. Используя инструменты библиотеки для работы с символьными многочленами (поиск точных корней, факторизация, символьное решение СЛАУ), модифицировать алгоритм из обязательной части.
  3. Найти точные символьные представления матрицы $J$ и матрицы перехода $P$.
  4. Генератор тестов: Написать скрипт для автоматической генерации тестовых данных.
    • Алгоритм генерации: Сначала сгенерируйте случайную Жорданову матрицу $J$ требуемой структуры и случайную невырожденную матрицу перехода $P$.
    • Затем вычислите целевую матрицу $A = P J P^{-1}$.
    • Подайте полученную матрицу $A$ на вход вашему алгоритму и сравните результат с исходной матрицей $J$.
    • Подсказка: Чтобы матрица $A$ состояла исключительно из целых чисел, матрица $P$ должна быть сгенерирована как унимодулярная матрица (с целыми элементами и определителем $\pm 1$).

Продвинутый уровень B

Численное приближение ЖНФ и Сингулярное разложение (SVD)

Вычисление ЖНФ является некорректно поставленной задачей вычислительной математики: малейшее изменение элементов матрицы из-за погрешности вычислений с плавающей точкой может разрушить структуру жордановых клеток.

Требуется найти "числовую" ЖНФ с заданной точностью $\epsilon = n \cdot 10^{-2}$ по фробениусовой норме, не используя символьные вычисления.

Задачи:

  1. Реализация SVD (Сингулярного разложения): Написать алгоритм SVD-разложения матрицы. Использовать встроенные функции SVD запрещено!
  2. Эффективный ранг ($\epsilon$-ранг): Классический метод Гаусса нестабилен для матриц, близких к вырожденным. Использовать полученное SVD для нахождения $\epsilon$-ранга матриц $(A - \lambda I)^k$. Ранг равен количеству сингулярных чисел $\sigma_i > \epsilon$.
  3. Поиск базиса ядра: С помощью SVD (матрица $V$ из разложения $A=U\Sigma V^T$) реализовать стабильное нахождение векторов из нуль-пространства $\ker(A - \lambda I)^k$.
  4. Проверить, что итоговое разложение удовлетворяет условию: $|A - P J P^{-1}| \le \epsilon$.
  5. Генератор тестов: Написать генератор тестов аналогично Части 2 (случайная $J$, случайная $P \to$ вычисление $A = P J P^{-1}$). Убедиться, что при внесении небольшого шума (в пределах $10^{-3}$) в элементы матрицы $A$, ваш алгоритм всё еще корректно восстанавливает структуру матрицы $J$.
