\Subsection{Описание программы}
Программа сначала устанавливает длину массива, загружает в текущий адрес адрес первого элемента массива. 
Затем циклически вычисляет битовую маску, где единица соответствует ненулевому элементу в массиве.

\Subsection{ОПИ и ОДЗ}

\textbf{Область представления}
\begin{itemize}
    \item 2EA, 2EB (адреса элементов массива) 11-разрядные числа, адрес памяти БЭВМ
    \item 2EC (длина массива) - 16-ти разрядное беззнаковое целое число
    \item 2ED (результат) - набор из 16 однобитовых логических значений
    \item 300-304 (элементы массива) - 16-ти разрядные знаковое целые числа
\end{itemize}

\textbf{Область допустимых значений}
\begin{itemize}
    \item 2EA (первый элемент) $\in$ [0; 2EA-длина массива] $\lor$ (300; 7FF-длина массива]
    \item 2EB (текущий элемент) $\in$ [первый элемент; первый элемент + длина массива - 1]
    \item 2EС (длина массива) $\in$ [0; 128]
    \item 2ED (результат) $\in$ [0; $2^{16}-1$]
    \item 300-304 (элементы массива) $\in$ [$-2^{15}$; $2^{15}-1$]
\end{itemize}
